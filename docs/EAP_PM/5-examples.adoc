= Examples

== EAP Packet Encoding and Decoding

The following example shows how an EAP packet can be encoded and decoded.

[source]
----
var PDU_EAP v_EAP_PDU;
var octetstring data;

data:= enc_PDU_EAP(v_EAP_PDU);

v_EAP_PDU := dec_PDU_EAP(data);
----
